# Daily Global News

Daily Global News is a lightweight Chrome extension that provides users with the **top 5 most significant global news stories** every day. Designed to keep you informed in a quick and convenient way, the extension curates news based on impact, credibility, diversity, and timeliness.

## Features
- Displays 5 important global news stories each day
- Covers multiple categories: Politics, Technology, Business, Science, Health, World
- Ensures geographic diversity (stories from different regions)
- Balanced content: includes major negative events and uplifting stories
- Easy-to-read popup directly from the Chrome toolbar
- Lightweight and fast, with no tracking or personal data collection

## Data Source
News stories are fetched from a curated JSON hosted on GitHub Pages, generated daily via an automated script using a language model API.
